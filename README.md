# La Boîte à Nono

Soundbox application made for fun under Ionic 3.

## Preview

Check [https://www.laboiteanono.fr](https://www.laboiteanono.fr)

## Install

```sh
$ npm i
```

## Serve (Dev)

```sh
$ npm run ionic:serve
```

## Build

```sh
$ npm run build -- --prod
```

## Contribute

If you want to contribute to the project you can do a pull request.
